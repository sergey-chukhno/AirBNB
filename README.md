# AirBNB Ruby Application

A Ruby-based AirBNB clone application.

## Getting Started

### Prerequisites

- Ruby 3.0 or higher
- Bundler

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   bundle install
   ```
